# Student Management System

Простое веб-приложение для управления списком студентов с возможностью добавления, удаления и редактирования записей.

## Технический стек

- Java 17
- Spring Boot 3.2.0
- SQLite
- jQuery

## Предварительные требования

Для сборки и запуска приложения вам потребуется:

1. **Java Development Kit (JDK)** версии 17 или выше
   - Проверить версию: `java -version`
   - Скачать можно здесь: [OpenJDK](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)

2. **Maven** версии 3.6.0 или выше
   - Проверить версию: `mvn -version`
   - Инструкция по установке: [Maven Installation Guide](https://maven.apache.org/install.html)

## Сборка проекта

1. Клонируйте репозиторий и перейдите в папку с pom.xml файлом:
```bash
git clone [URL репозитория]
cd student-management
```

2. Соберите проект с помощью Maven:
```bash
mvn clean package
```

После успешной сборки, JAR файл будет создан в директории `target/`.

## Запуск приложения

1. Перейдите в директорию с JAR файлом:
```bash
cd target
```

2. Запустите приложение:
```bash
java -jar student-management-1.0.0.jar
```

После запуска:
- Приложение будет доступно по адресу: `http://localhost:8080`
- База данных будет автоматически создана в директории `data/`

## Использование приложения

1. Откройте браузер и перейдите по адресу: `http://localhost:8080`

2. В приложении доступны следующие функции:
   - Добавление нового студента
   - Просмотр списка всех студентов
   - Редактирование информации о студенте
   - Удаление студента
   - Поиск по списку студентов
